
# Zoomeze Dog Play Zone Website

This is a ready-to-upload React + Vite website starter for Zoomeze Dog Play Zone.

## What this package includes
- Home page with your logo
- Indoor and outdoor photos
- Pricing and opening hours
- Draft booking page layout
- Contact details

## Important
The booking form on this website is a visual draft only.
To take real bookings and payments, connect it later to a booking/payment system such as:
- SimplyBook.me
- Bookeo
- Stripe checkout with a booking tool

## Upload to GitHub
1. Create a new GitHub repository called `zoomeze-website`
2. Open the repository
3. Click **Add file**
4. Click **Upload files**
5. Drag **all files from this folder** into GitHub
6. Click **Commit changes**

## Deploy on Vercel
1. Create a Vercel account and sign in with GitHub
2. Click **Add New... > Project**
3. Import the `zoomeze-website` repository
4. Click **Deploy**
5. Once live, add your custom domain in Vercel:
   - `zoomezeplayhouse.com`

## Connect the domain
In Vercel:
- Go to your project
- Settings
- Domains
- Add `zoomezeplayhouse.com`

Vercel will then tell you what DNS record to add in Namecheap.

## Local preview (optional)
If you want to preview it on your own machine:
```bash
npm install
npm run dev
```
